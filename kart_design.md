# 🏎️ 머슴카트 — 봇 레이싱 아이템전

## 컨셉
봇끼리 2D 탑다운 레이싱 + 아이템전. 관전자가 보는 엔터테인먼트.

## 트랙
- 타원형 기본 트랙 (캔버스 800x600)
- 체크포인트 시스템 (순위 계산)
- 3바퀴 경주
- 트랙 위 아이템 박스 랜덤 배치

## 봇 (4~8대)
| 이름 | 성격 | 색상 |
|------|------|------|
| 🔴 불꽃돌쇠 | 공격형 (미사일 선호) | 빨강 |
| 🔵 빙하돌쇠 | 방어형 (방패 선호) | 파랑 |
| 🟢 질풍돌쇠 | 스피드형 (부스트 선호) | 초록 |
| 🟡 함정돌쇠 | 트랩형 (바나나 선호) | 노랑 |
| 🟣 번개돌쇠 | 역전형 (번개 선호) | 보라 |
| 🟠 도박돌쇠 | 랜덤형 (아무거나) | 주황 |

## 아이템 (6종)
| 아이템 | 효과 | 확률 (선두) | 확률 (후미) |
|--------|------|------------|------------|
| 🚀 미사일 | 전방 직선 공격, 맞으면 1초 스턴 | 30% | 10% |
| 🍌 바나나 | 뒤에 설치, 밟으면 0.8초 스핀 | 25% | 10% |
| ⚡ 부스트 | 2초간 속도 1.5배 | 15% | 25% |
| 🛡️ 방패 | 3초간 공격 무효화 | 10% | 15% |
| ⚡ 번개 | 전체 0.5초 감속 (자신 제외) | 5% | 25% |
| 🌟 슈퍼스타 | 5초 무적+가속 (꼴찌 전용) | 0% | 15% |

**순위 역보정**: 뒤처진 봇일수록 강한 아이템 나옴 (마리오카트 공식)

## 물리
- 속도: 기본 3px/tick, 부스트 4.5px/tick
- 가속/감속: 0.1px/tick²
- 드리프트: 코너에서 감속 없이 통과 (확률적)
- 충돌: 벽 반사, 봇끼리 밀치기
- 틱: 60ms (약 16fps 시뮬레이션)

## 관전 UI
- 미니맵 전체 트랙 뷰
- 순위판 (실시간)
- 아이템 사용 시 이펙트 + 해설
- "🚀 불꽃돌쇠의 미사일이 질풍돌쇠를 명중!"
- 결승선 통과 시 슬로모션 연출

## API (외부 봇 참여용)
- POST /api/kart/join — 레이스 참가
- GET /api/kart/state — 현재 상태 (위치, 속도, 아이템, 순위)
- POST /api/kart/action — 아이템 사용, 가속/감속
- WS /ws/kart — 실시간 관전
